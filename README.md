# White Rock - Real Estate Management System

**White Rock** is a comprehensive real estate platform designed to streamline the interaction between property agents and potential buyers. This project was developed as a final project for the **Higher Technician in Web Application Development (DAW)** vocational degree.

---

## 🎓 Academic Context
This project serves as a demonstration of full-stack development skills acquired during the **DAW (Desarrollo de Aplicaciones Web)** program. It focuses on implementing complex business logic, role-based access control, and database relational integrity.

---

## 🚀 Core Features

### For Users (Clients)
- **Property Catalog:** Browse and search through available houses and apartments.
- **Detailed View:** Access specific details, pricing, and features of each property.
- **Visit Requests:** Integrated system to request physical visits to specific properties.

### For Agents (Administrators)
- **Property Management (CRUD):** Full control to create, update, and delete property listings.
- **Lead Management:** Monitor and manage visit requests sent by users.
- **Inventory Control:** Track the status and availability of the real estate portfolio.

---

## 🛠️ Tech Stack

The application is built using a classic robust architecture:

-   **Backend:** [Laravel](https://laravel.com/) (PHP Framework).
-   **Database:** [MySQL](https://www.mysql.com/) (Relational database management).
-   **Frontend:** Blade Templating Engine with CSS/Bootstrap.
-   **ORM:** Eloquent for database interactions and relationship management.
